# **App Name**: Recovery Engine

## Core Features:

- Teacher/Admin Secure Authentication: Secure login for teachers and administrators to access their respective portals and manage classes, students, and materials.
- Course & Study Material Management: Admins can create courses, upload lecture plans (manual entry or PDF), and associated study materials (PDF/PPTX) organized by modules or topics.
- Student Roster Management: Admins can upload student details (name, registration number, email) via Excel/CSV sheets for specific courses, automatically populating the student database.
- Attendance Recording & Absentee Detection: Teachers can manually mark student attendance or upload attendance sheets (Excel/PDF). The system automatically identifies absent students for each lecture.
- AI-Powered Lecture Summary & Notes Tool: Utilizes machine learning to detect headings in uploaded study materials (PDF/PPTX) and, based on the lecture planner, generate a concise lecture summary and identify relevant sections/PDF notes for missed topics.
- Automated Absentee Email Notification: Automatically sends personalized emails to absent students containing the AI-generated lecture summary and relevant PDF notes/links for the missed class.
- Absentee Report Dashboard: Admins can view and download comprehensive reports of absent students for any given lecture, including their details and a summary of content sent.

## Style Guidelines:

- Color scheme: Light. Primary color: A structured, modern blue (#2E5CB3). Background color: A very subtle, desaturated blue-gray (#EEF1F6). Accent color: A refreshing, informative teal (#51AEC6) for highlights and calls to action.
- Headings: 'Space Grotesk' (sans-serif) for a contemporary, tech-forward feel. Body text: 'Inter' (sans-serif) for clean, objective readability suitable for academic content.
- Use a set of clear, outline-style icons to represent actions like upload, email, report, and student management, maintaining a professional and modern aesthetic.
- Employ a clean, modular layout with clear separation between content areas, favoring dashboards and data tables for administrative views to enhance information hierarchy and usability.
- Implement subtle transition animations for content loading and state changes, providing smooth user feedback without being distracting. For instance, a slight fade-in on new data or a gentle pulse on notification triggers.
